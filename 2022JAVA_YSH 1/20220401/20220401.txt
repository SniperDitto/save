
클래스
 - 속성(필드)
    - 해당 객체가 갖는 특징을 속성으로 표현한다
 - 생성자
    - 클래스가 인스턴스화 될 때(새로운 객체 new로 생성시)자동으로 실행
    - 리턴값 없이 클래스 이름과 똑같이 만드는 메서드
    - 오버로딩을 이용해 여러 개의 생성자를 다른 매개변수로 만들 수 있다
 - 메서드
    - 해당 객체가 한 행위를 메서드로 표현한다
        ex)상품의 갯수 체크 메서드는 상품 클래스에 있어야 함
    - 오버로딩 : 같은 이름의 메서드를 다른 매개변수로 똑같이 만드는 것, 서로 다른 메서드로 취급됨
